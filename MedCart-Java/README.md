# MedCart-Java

Console-based Java medical store. View medicines, add to cart, view cart, and checkout.

## How to run (local)
javac MedCart.java
java MedCart

## Demo keys to press
1 → show medicines → enter 2 to add Ibuprofen  
2 → view cart  
3 → checkout  
4 → exit
